# 4. 어댑터뷰

## 학습목표
- 리스트뷰와 그리드뷰 활용법을 이해한다
- 갤러리와 스피너 사용법을 익힌다

---

## Section 01. 리스트뷰와 그리드뷰

### 01 어댑터뷰

어댑터뷰 계층 구조:
```
java.lang.Object
  └ android.view.View
      └ android.widget.ViewGroup
          └ android.widget.AdapterView
              ├ android.widget.AbsListView
              │   ├ android.widget.GridView
              │   └ android.widget.ListView
              │       └ android.widget.ExpandableListView
              └ android.widget.AbsSpinner
                  ├ android.widget.Spinner
                  └ android.widget.Gallery
```

- **어댑터뷰 종류**: `ListView`, `GridView`, `ExpandableListView`, `Spinner`, `Gallery`
- 어댑터뷰의 모양 설정 및 데이터 주입에 **`ArrayAdapter<T>`** 클래스를 함께 사용

---

### 02 리스트뷰 (ListView)

데이터를 리스트 모양으로 보여주고 하나를 선택하는 용도

#### 리스트뷰 만들기 순서

1. 리스트뷰에 나열할 내용을 `String` 배열로 준비
2. `XML`에 `<ListView>` 생성, Kotlin에서 변수로 연결
3. `ArrayAdapter<String>` 생성 (모양 + 데이터 지정)
4. 리스트뷰에 어댑터 적용
5. 클릭 이벤트 리스너 정의

**XML:**
```xml
<LinearLayout>
    <ListView
        android:id="@+id/listView1"
        android:layout_width="match_parent"
        android:layout_height="wrap_content" />
</LinearLayout>
```

**Kotlin:**
```kotlin
val mid = arrayOf("히어로즈", "24시", "로스트", "로스트룸", "스몰빌", ...)

val list = findViewById<View>(R.id.listView1) as ListView

val adapter: ArrayAdapter<String> = ArrayAdapter(this,
    android.R.layout.simple_list_item_1, mid)
list.adapter = adapter

list.setOnItemClickListener { parent, view, position, id ->
    Toast.makeText(applicationContext, mid[position], Toast.LENGTH_SHORT).show()
}
```

#### 다양한 모양 설정

| 레이아웃 상수 | 모양 |
|---|---|
| `simple_list_item_1` | 기본 텍스트 목록 |
| `simple_list_item_multiple_choice` | 체크박스 목록 |
| `simple_list_item_single_choice` | 라디오버튼 목록 |

```kotlin
// 체크박스 모양
val adapter = ArrayAdapter<String>(this,
    android.R.layout.simple_list_item_multiple_choice, mid)
list.choiceMode = ListView.CHOICE_MODE_MULTIPLE
list.adapter = adapter
```

#### 동적 추가·삭제

```kotlin
val midList = ArrayList<String>()
val adapter = ArrayAdapter(this, android.R.layout.simple_list_item_1, midList)
list.adapter = adapter

// 추가
btnAdd.setOnClickListener {
    midList.add(edtItem.text.toString())
    adapter.notifyDataSetChanged()  // 어댑터에 변경 알림
}

// 삭제 (롱클릭)
list.setOnItemLongClickListener { parent, view, position, id ->
    midList.removeAt(position)
    adapter.notifyDataSetChanged()
    false
}
```

### 그리드뷰 (GridView)

이미지 등을 격자 형태로 배치

**XML:**
```xml
<GridView
    android:id="@+id/gridView1"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:gravity="center"
    android:numColumns="4" />
```

#### BaseAdapter를 상속받아 커스텀 어댑터 구현

```kotlin
// onCreate() 안
val gv = findViewById<View>(R.id.gridView1) as GridView
val gAdapter = MyGridAdapter(this)
gv.adapter = gAdapter

// 내부 클래스
inner class MyGridAdapter(var context: Context) : BaseAdapter() {

    var posterID = arrayOf(
        R.drawable.mov01, R.drawable.mov02, R.drawable.mov03,
        // ...
    )

    override fun getCount(): Int = posterID.size

    override fun getView(p0: Int, p1: View?, p2: ViewGroup?): View {
        val imageview = ImageView(context)
        imageview.layoutParams = ViewGroup.LayoutParams(200, 300)
        imageview.scaleType = ImageView.ScaleType.FIT_CENTER
        imageview.setPadding(5, 5, 5, 5)
        imageview.setImageResource(posterID[p0])
        return imageview
    }

    override fun getItem(p0: Int): Any = 0
    override fun getItemId(p0: Int): Long = 0
}
```

#### 클릭 시 대화상자로 큰 이미지 표시

```kotlin
imageview.setOnClickListener {
    val dialogView = View.inflate(this@MainActivity, R.layout.dialog, null)
    val dlg = AlertDialog.Builder(this@MainActivity)
    val ivPoster = dialogView.findViewById<ImageView>(R.id.ivPoster)
    ivPoster.setImageResource(posterID[p0])
    dlg.setTitle("큰 포스터")
    dlg.setIcon(R.drawable.ic_launcher)
    dlg.setView(dialogView)
    dlg.setNegativeButton("닫기", null)
    dlg.show()
}
```

## Section 02. 갤러리와 스피너

### 01 갤러리 (Gallery)

- 이미지를 좌우로 스크롤하여 보여주는 뷰
- 이미지 클릭 시 별도 동작은 Kotlin 코드로 추가 필요

**XML:**
```xml
<LinearLayout>
    <Gallery
        android:id="@+id/gallery1"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:spacing="5dp" />
    <ImageView
        android:id="@+id/ivPoster"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:padding="20dp" />
</LinearLayout>
```

**Kotlin:**
```kotlin
val gallery = findViewById<Gallery>(R.id.gallery1)
val galAdapter = MyGalleryAdapter(this)
gallery.adapter = galAdapter

inner class MyGalleryAdapter(var context: Context) : BaseAdapter() {

    var posterID = arrayOf(
        R.drawable.mov11, R.drawable.mov12, // ...
    )

    override fun getCount(): Int = posterID.size

    override fun getView(p0: Int, p1: View?, p2: ViewGroup?): View {
        val imageview = ImageView(context)
        imageview.layoutParams = Gallery.LayoutParams(200, 300)
        imageview.scaleType = ImageView.ScaleType.FIT_CENTER
        imageview.setPadding(5, 5, 5, 5)
        imageview.setImageResource(posterID[p0])

        // 터치 시 아래 이미지뷰에 큰 이미지 표시
        imageview.setOnTouchListener { v, event ->
            val ivPoster = findViewById<ImageView>(R.id.ivPoster)
            ivPoster.scaleType = ImageView.ScaleType.FIT_CENTER
            ivPoster.setImageResource(posterID[p0])
            false
        }
        return imageview
    }

    override fun getItem(p0: Int): Any = 0
    override fun getItemId(p0: Int): Long = 0
}
```

### 02 스피너 (Spinner)

- PC의 드롭다운 박스와 유사
- 화면이 작은 스마트폰에서 여러 항목 중 하나를 선택할 때 사용

**XML:**
```xml
<Spinner
    android:id="@+id/spinner1"
    android:layout_width="match_parent"
    android:layout_height="wrap_content" />
```

**Kotlin:**
```kotlin
val movie = arrayOf("콩푸팬더", "짱구는 못말려", "아저씨", "아바타", "대부", ...)

val spinner = findViewById<Spinner>(R.id.spinner1)

val adapter: ArrayAdapter<String>
adapter = ArrayAdapter(this, android.R.layout.simple_spinner_item, movie)
spinner.adapter = adapter

// 선택 이벤트
spinner.onItemSelectedListener = object : AdapterView.OnItemSelectedListener {
    override fun onItemSelected(parent: AdapterView<*>, view: View, position: Int, id: Long) {
        // movie[position] 사용
    }
    override fun onNothingSelected(parent: AdapterView<*>) {}
}
```

## 핵심 정리

| 뷰 | 특징 | 어댑터 |
|---|---|---|
| **ListView** | 세로 스크롤 목록 | `ArrayAdapter` |
| **GridView** | 격자형 배치 | `BaseAdapter` (커스텀) |
| **Gallery** | 가로 스크롤 이미지 | `BaseAdapter` (커스텀) |
| **Spinner** | 드롭다운 선택 | `ArrayAdapter` |

**동적 변경 시 필수:** `adapter.notifyDataSetChanged()` 호출

**커스텀 어댑터 필수 구현 메서드:**
- `getCount()` : 항목 수 반환
- `getView()` : 각 항목의 뷰 반환
- `getItem()` : 항목 데이터 반환
- `getItemId()` : 항목 ID 반환
