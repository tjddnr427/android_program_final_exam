# 1. 고급 위젯 다루기

## 학습목표
- 고급 위젯을 다루는 방법을 익힌다.
- 뷰 컨테이너와 그 응용법을 이해한다.

---

## SECTION 01. 고급 위젯

### 01 날짜와 시간 관련 위젯

#### 아날로그시계 / 디지털시계
- 화면에 시간을 표시하는 위젯
- `View` 클래스에서 상속받기 때문에 `background` 속성 등 설정 가능
- `DigitalClock`은 `textColor` 같은 속성도 설정 가능

**계층도**
```
java.lang.Object
  └ android.view.View
      └ android.widget.AnalogClock
      └ android.widget.TextView
          └ android.widget.DigitalClock
```

**예제 XML 코드**
```xml
<LinearLayout>
    <AnalogClock
        android:layout_width="match_parent"
        android:layout_height="wrap_content" />
    <DigitalClock
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:gravity="center" />
</LinearLayout>
```

---

#### 크로노미터 (Chronometer)
- 타이머 형식의 위젯, 시간 측정에 주로 사용
- 주요 메소드: `start()`, `stop()`, `reset()`

**계층도**
```
java.lang.Object
  └ android.view.View
      └ android.widget.TextView
          └ android.widget.Chronometer
```

**예제 XML 코드**
```xml
<Chronometer
    android:id="@+id/chronometer1"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:format="시간 측정 : %s"
    android:gravity="center"
    android:textSize="30dp" />
```

---

#### 타임피커 / 데이트피커 / 캘린더뷰
- **TimePicker**: 시간 표시와 조절
- **DatePicker**: 날짜 표시와 조절
- **CalendarView**: 날짜 표시와 조절 (달력 형태)
  - `showWeekNumber`: 현재 몇 주 차인지 표시 (기본값 `true`, `false` 권장)

> 시간과 날짜는 **대화상자(다이얼로그 박스)** 로 표현하는 것을 권장.  
> Kotlin 코드에서 `TimePickerDialog`, `DatePickerDialog` 클래스 사용.

#### 실습 6-1: 날짜/시간 예약 앱 만들기

**기능 개요**
- 타이머 기능: `<예약 시작>`, `<예약완료>` 클릭 시 크로노미터 동작
- 날짜/시간 설정: `<날짜 설정>`, `<시간 설정>` 클릭 시 예약 날짜와 시간 변경
- `<예약완료>` 클릭 시 설정한 날짜와 시간 결정

**레이아웃 구성 (activity_main.xml)**

| 위젯 | ID |
|------|-----|
| Chronometer | chronometer1 |
| Button (예약 시작) | btnStart |
| RadioButton (날짜 설정) | rdoCal |
| RadioButton (시간 설정) | rdoTime |
| CalendarView | calendarView1 |
| TimePicker | timePicker1 |
| Button (예약완료) | btnEnd |
| TextView (년) | tvYear |
| TextView (월) | tvMonth |
| TextView (일) | tvDay |
| TextView (시) | tvHour |
| TextView (분) | tvMinute |

**주요 Kotlin 코드**
```kotlin
// 라디오버튼 클릭 이벤트
rdoCal.setOnClickListener {
    tPicker.visibility = View.INVISIBLE
    calView.visibility = View.VISIBLE
}
rdoTime.setOnClickListener {
    tPicker.visibility = View.VISIBLE
    calView.visibility = View.INVISIBLE
}

// 크로노미터 시작/정지
btnStart.setOnClickListener {
    chrono.base = SystemClock.elapsedRealtime()
    chrono.start()
    chrono.setTextColor(Color.RED)
}
btnEnd.setOnClickListener {
    chrono.stop()
    chrono.setTextColor(Color.BLUE)
}

// 캘린더뷰 날짜 변경 리스너
calView.setOnDateChangeListener { view, year, month, dayOfMonth ->
    selectYear = year
    selectMonth = month + 1
    selectDay = dayOfMonth
}

// 예약완료 시 텍스트뷰에 날짜/시간 표시
tvYear.text = Integer.toString(selectYear)
tvMonth.text = Integer.toString(selectMonth)
tvDay.text = Integer.toString(selectDay)
tvHour.text = Integer.toString(tPicker.currentHour)
tvMinute.text = Integer.toString(tPicker.currentMinute)
```

### 02 기타 위젯

#### 자동완성텍스트뷰 / 멀티자동완성텍스트뷰
- **AutoCompleteTextView**: 단어 1개가 자동 완성
- **MultiAutoCompleteTextView**: 쉼표(,)로 구분하여 여러 단어 자동 완성
- 자동 완성 단어는 Kotlin 코드에서 배열로 설정, `setAdapter()` 메소드 이용

**계층도**
```
java.lang.Object
  └ android.view.View
      └ android.widget.TextView
          └ android.widget.EditText
              └ android.widget.EditText.AutoCompleteTextView
                  └ android.widget.EditText.MultiAutoCompleteTextView
```

**Kotlin 코드 예제**
```kotlin
var items = arrayOf("CSI-뉴욕", "CSI-라스베가스", "CSI-마이애미", "Friends", "Fringe", "Lost")

var auto = findViewById<AutoCompleteTextView>(R.id.autoCompleteTextView1)
var adapter = ArrayAdapter(this, android.R.layout.simple_dropdown_item_1line, items)
auto.setAdapter(adapter)

var multi = findViewById<MultiAutoCompleteTextView>(R.id.multiAutoCompleteTextView1)
var token = MultiAutoCompleteTextView.CommaTokenizer()
multi.setTokenizer(token)
multi.setAdapter(adapter)
```

#### 프로그레스바 / 시크바 / 레이팅바

**계층도**
```
java.lang.Object
  └ android.view.View
      └ android.widget.ProgressBar
      └ android.widget.AbsSeekBar
          └ android.widget.RatingBar
          └ android.widget.SeekBar
```

**ProgressBar**
- 작업 진행 상태를 바(bar) 또는 원 형태로 제공
- 주요 XML 속성: `max`, `progress`, `secondaryProgress`

**SeekBar**
- 프로그레스바의 하위 클래스, 사용자가 터치로 임의 조절 가능
- 음량 조절, 동영상 재생 위치 지정 등에 활용

**RatingBar**
- 진행 상태를 별 모양으로 표시
- 주요 XML 속성: `numStars`, `rating`, `stepSize`

**XML 코드 예제**
```xml
<ProgressBar
    style="?android:attr/progressBarStyleHorizontal"
    android:max="100"
    android:progress="20"
    android:secondaryProgress="50" />
<SeekBar
    android:progress="20" />
<RatingBar
    android:numStars="5"
    android:rating="1.5"
    android:stepSize="0.5" />
```

## SECTION 02. 뷰 컨테이너

### 01 간단한 기능의 뷰 컨테이너

#### 스크롤뷰 (ScrollView)
- 위젯이나 레이아웃이 화면에 넘칠 때 스크롤 효과 제공
- 수직(위아래) 스크롤 담당
- 수평 스크롤은 `HorizontalScrollView` 사용
- 스크롤뷰 안에는 **단 하나의 위젯**만 넣을 수 있음
  - 주로 `ScrollView > LinearLayout > 여러 위젯` 패턴 사용

**XML 코드 예제**
```xml
<ScrollView
    android:layout_width="fill_parent"
    android:layout_height="fill_parent">
    <LinearLayout
        android:layout_width="fill_parent"
        android:layout_height="fill_parent"
        android:orientation="vertical">
        <Button android:layout_height="100dp" android:text="버튼 1" />
        <!-- 버튼 추가 -->
    </LinearLayout>
</ScrollView>
```

---

#### 슬라이딩드로어 (SlidingDrawer)
- 위젯을 서랍처럼 열어서 보여주거나 닫아서 감춤

**구조 형태**
```xml
<SlidingDrawer android:handle="@+id/handle" android:content="@+id/content">
    <Button android:id="@+id/handle" android:text="서랍 손잡이" />
    <LinearLayout android:id="@+id/content">
        <!-- 서랍 내부 위젯 -->
    </LinearLayout>
</SlidingDrawer>
```

**규칙**
- `handle` 속성값 = 손잡이 역할 버튼(또는 이미지뷰)의 `id`
- `content` 속성값 = 서랍 내부 레이아웃의 `id`

### 02 복잡한 기능의 뷰 컨테이너

#### 뷰플리퍼 (ViewFlipper)
- 여러 위젯을 배치하고 화면을 좌우로 밀어서 하나씩 보여주는 뷰 컨테이너

**계층도**
```
java.lang.Object
  └ android.view.View
      └ android.widget.ViewGroup
          └ android.widget.FrameLayout
              └ android.widget.ViewAnimator
                  └ android.widget.ViewFlipper
```

**구조 형태**
```xml
<LinearLayout>
    <LinearLayout>
        <!-- 이전/다음 화면 전환 버튼 -->
    </LinearLayout>
    <ViewFlipper android:id="@+id/viewFlipper1">
        <LinearLayout android:background="#ff0000"> <!-- 화면 1 --> </LinearLayout>
        <LinearLayout android:background="#00ff00"> <!-- 화면 2 --> </LinearLayout>
        <LinearLayout android:background="#0000ff"> <!-- 화면 3 --> </LinearLayout>
    </ViewFlipper>
</LinearLayout>
```

**Kotlin 코드**
```kotlin
btnPrev.setOnClickListener {
    vFlipper.showPrevious()
}
btnNext.setOnClickListener {
    vFlipper.showNext()
}
```

> **자동 슬라이드**: `startFlipping()`, `stopFlipping()`, `setFlipInterval(밀리초)` 사용

#### 탭호스트 (TabHost)
- 여러 탭을 두고 각 탭 클릭 시 해당 화면이 나오도록 설정하는 뷰 컨테이너

**구조 형태**
```xml
<TabHost android:id="@android:id/tabhost">
    <LinearLayout>
        <TabWidget android:id="@android:id/tabs" />
        <FrameLayout android:id="@android:id/tabcontent">
            <!-- 각 탭 화면(레이아웃) 3개 삽입 -->
        </FrameLayout>
    </LinearLayout>
</TabHost>
```

> **주의**: `tabhost`, `tabs`, `tabcontent` ID는 변경하지 말고 그대로 사용해야 함.

**Kotlin 코드 (탭스펙 생성)**
```kotlin
var tabHost = this.tabHost

var tabSpec1 = tabHost.newTabSpec("TAB1").setIndicator("탭에 출력될 글자")
tabSpec1.setContent(R.id.tab1)
tabHost.addTab(tabSpec1)
```

**전체 예제**
```kotlin
@Suppress("deprecation")
class MainActivity : TabActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        var tabHost = this.tabHost

        var tabSpecSong = tabHost.newTabSpec("SONG").setIndicator("음악별")
        tabSpecSong.setContent(R.id.tabSong)
        tabHost.addTab(tabSpecSong)

        // Artist, Album 탭도 동일한 방식으로 추가

        tabHost.currentTab = 0
    }
}
```

#### 액션바와 프래그먼트

**액션바 (ActionBar)**
- 허니콤(Android 3.0, API 11)부터 지원
- 태블릿 등 대형 화면에서 여러 화면을 사용하기 위해 고안
- 액션바 구현 시 **프래그먼트(Fragment)** 사용 권장

**프래그먼트 (Fragment)**
- 허니콤에서 소개된 기능
- 액티비티보다 작은 단위의 화면
- 대형 화면: 액티비티 화면 분할 표현
- 소형 화면: 실행 중 화면을 동적으로 추가/제거

**액션바에 탭 추가하는 형식**
```kotlin
var bar = this.supportActionBar
bar!!.navigationMode = ActionBar.NAVIGATION_MODE_TABS

tabSong = bar.newTab()
tabSong.text = "음악별"
tabSong.setTabListener(this)
bar.addTab(tabSong)
```

**프래그먼트 클래스**
```kotlin
class MyTabFragment : androidx.fragment.app.Fragment() {
    var tabName: String? = null

    override fun onCreateView(inflater: LayoutInflater, container: ViewGroup?,
                              savedInstanceState: Bundle?): View? {
        var baseLayout = LinearLayout(super.getActivity())
        baseLayout.orientation = LinearLayout.VERTICAL

        if (tabName === "음악별") baseLayout.setBackgroundColor(Color.RED)
        if (tabName === "가수별") baseLayout.setBackgroundColor(Color.GREEN)
        if (tabName === "앨범별") baseLayout.setBackgroundColor(Color.BLUE)

        return baseLayout
    }
}
```
