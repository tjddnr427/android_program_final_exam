## SECTION 01. 메뉴

### 메뉴 종류

| 종류 | 설명 |
|------|------|
| 옵션 메뉴 (option menu) | 메뉴 버튼 또는 화면 오른쪽 위 아이콘 클릭 시 하단에 나오는 메뉴 |
| 컨텍스트 메뉴 (context menu) | 위젯 등을 롱클릭하면 나오는 메뉴, 화면 중앙에 출력 |

---

### 01 XML을 이용한 옵션 메뉴

**설정 순서**

1. 메뉴 폴더 생성 및 메뉴 XML 파일 생성·편집 (메뉴 코딩)
2. Kotlin 코딩: `onCreateOptionsMenu()` 메소드 오버라이딩 (메뉴 파일 등록)
3. Kotlin 코딩: `onOptionsItemSelected()` 메소드 오버라이딩 (메뉴 선택 시 동작 코딩)

**메뉴 XML 파일 형식**
```xml
<menu>
    <item
        android:id="@+id/항목1아이디"
        android:title="항목1 제목"/>
    <item
        android:id="@+id/항목2아이디"
        android:title="항목2 제목"/>
</menu>
```

**서브메뉴 추가**
```xml
<item android:title="버튼 변경 >>">
    <menu>
        <item android:id="@+id/subRotate" android:title="버튼 45도 회전"/>
        <item android:id="@+id/subSize" android:title="버튼 2배 확대"/>
    </menu>
</item>
```

**onCreateOptionsMenu() 형식**
```kotlin
override fun onCreateOptionsMenu(menu: Menu?) : Boolean {
    super.onCreateOptionsMenu(menu)
    var mInflater = menuInflater
    mInflater.inflate(R.menu.메뉴XML아이디, menu)
    return true
}
```

**onOptionsItemSelected() 형식**
```kotlin
override fun onOptionsItemSelected(item: MenuItem) : Boolean {
    when (item.itemId) {
        R.id.항목1아이디 -> {
            // 항목1 선택 시 실행할 코드
            return true
        }
        R.id.항목2아이디 -> {
            // 항목2 선택 시 실행할 코드
            return true
        }
    }
    return false
}
```

#### 실습 7-1: 배경색 변경 앱 만들기

**레이아웃 구성**
```xml
<LinearLayout android:id="@+id/baseLayout">
    <TextView android:text="오른쪽 위 메뉴 버튼을 누르세요" />
    <Button android:id="@+id/button1" android:text="이건 버튼" />
</LinearLayout>
```

**menu1.xml**
```xml
<menu>
    <item android:id="@+id/itemRed" android:title="배경색(빨강)"/>
    <item android:id="@+id/itemGreen" android:title="배경색(초록)"/>
    <item android:id="@+id/itemBlue" android:title="배경색(파랑)"/>
    <item android:title="버튼 변경 >>">
        <menu>
            <item android:id="@+id/subRotate" android:title="버튼 45도 회전"/>
            <item android:id="@+id/subSize" android:title="버튼 2배 확대"/>
        </menu>
    </item>
</menu>
```

**Kotlin 코드 (선택 처리)**
```kotlin
override fun onOptionsItemSelected(item: MenuItem) : Boolean {
    when (item.itemId) {
        R.id.itemRed -> {
            baseLayout.setBackgroundColor(Color.RED)
            return true
        }
        R.id.subRotate -> {
            button1.rotation = 45f
            return true
        }
        R.id.subSize -> {
            button1.scaleX = 2f
            return true
        }
    }
    return false
}
```

### 02 Kotlin 코드만 이용한 옵션 메뉴

- `onCreateOptionsMenu()` 안에서 `menu.add()` 메소드로 직접 항목 추가
- `onOptionsItemSelected()`의 when 문을 id 번호로 변경

```kotlin
override fun onCreateOptionsMenu(menu: Menu?) : Boolean {
    super.onCreateOptionsMenu(menu)

    menu!!.add(0, 1, 0, "배경색 (빨강)")
    menu!!.add(0, 2, 0, "배경색 (초록)")
    menu!!.add(0, 3, 0, "배경색 (파랑)")

    var sMenu : SubMenu = menu.addSubMenu("버튼 변경 >>")
    sMenu.add(0, 4, 0, "버튼 45도 회전")
    sMenu.add(0, 5, 0, "버튼 2배 확대")

    return true
}

override fun onOptionsItemSelected(item: MenuItem): Boolean {
    when (item.itemId) {
        1 -> {
            baseLayout.setBackgroundColor(Color.RED)
            return true
        }
        // 2, 3, 4, 5 동일한 방식
    }
    return false
}
```

> **Hint**: 메뉴 XML에서 여러 항목을 묶으려면 `<group android:checkableBehavior="single">...</group>` 사용

### 03 XML을 이용한 컨텍스트 메뉴

**설정 순서**

1. 메뉴 폴더 생성 및 위젯별 메뉴 XML 파일 생성·편집
2. `onCreate()` 안에 `registerForContextMenu(위젯)`으로 등록
3. `onCreateContextMenu()` 메소드 오버라이딩 (메뉴 파일 등록)
4. `onContextItemSelected()` 메소드 오버라이딩 (메뉴 선택 시 동작 코딩)

**onCreate() 등록**
```kotlin
registerForContextMenu(button1)
registerForContextMenu(button2)
```

**onCreateContextMenu() 형식**
```kotlin
override fun onCreateContextMenu(menu: ContextMenu?, v: View?,
                                 menuInfo: ContextMenu.ContextMenuInfo?) {
    super.onCreateContextMenu(menu, v, menuInfo)
    var mInflater = this.menuInflater
    if (v === button1) {
        menu!!.setHeaderTitle("배경색 변경")
        mInflater.inflate(R.menu.menu1, menu)
    }
    if (v === button2) {
        mInflater.inflate(R.menu.menu2, menu)
    }
}
```

## SECTION 02. 토스트

### 토스트 (Toast)
- 화면에 잠깐 나타났다 사라지는 메시지
- 사용자가 인식해야 할 작은 메시지 또는 디버깅 용도

**기본 사용 형식**
```kotlin
Toast.makeText(Context, "메시지", duration).show()
```

| 파라미터 | 설명 |
|----------|------|
| `context` | 현재 액티비티를 나타내는 `this` (람다식 내부에서는 `applicationContext`) |
| `duration` | `Toast.LENGTH_LONG` 또는 `Toast.LENGTH_SHORT` |

**위치 변경 (setGravity)**
```kotlin
// 기본 위치: 화면 중앙 하단
Toast.setGravity(gravity, xOffset, yOffset)
```

**랜덤 위치 토스트 예제**
```kotlin
button1.setOnClickListener {
    var tMsg = Toast.makeText(applicationContext, "토스트 연습", Toast.LENGTH_SHORT)

    var display = (getSystemService(Context.WINDOW_SERVICE) as WindowManager).defaultDisplay
    var xOffset = (Math.random() * display.width).toInt()
    var yOffset = (Math.random() * display.height).toInt()

    tMsg.setGravity(Gravity.TOP or Gravity.LEFT, xOffset, yOffset)
    tMsg.show()
}
```

## SECTION 03. 대화상자

### 01 기본 대화상자

- **Dialog**: 화면에 메시지를 나타낸 후 확인/취소 같은 사용자 선택을 받아들이는 UI
- 토스트보다 강력한 메시지 표시나 계속 진행 여부 선택 시 사용

**설정 순서**

1. `AlertDialog.Builder` 클래스로 대화상자 생성
2. `setTitle()`, `setMessage()`, `setIcon()` 등으로 꾸미기
3. `setPositiveButton()`, `setNegativeButton()` 등으로 버튼 추가
4. `show()`로 화면 출력

**기본 대화상자**
```kotlin
button1.setOnClickListener {
    var dlg = AlertDialog.Builder(this@MainActivity)
    dlg.setTitle("제목입니다")
    dlg.setMessage("이곳이 내용입니다")
    dlg.setIcon(R.mipmap.ic_launcher)
    dlg.show()
}
```

**버튼 1개 (확인만)**
```kotlin
dlg.setPositiveButton("확인", null)
dlg.show()
```

**버튼 클릭 시 동작 추가**
```kotlin
dlg.setPositiveButton("확인") { dialog, which ->
    Toast.makeText(this@MainActivity, "확인을 눌렀네요", Toast.LENGTH_SHORT).show()
}
```

**확인 + 취소 버튼**
```kotlin
dlg.setPositiveButton("확인", null)
dlg.setNegativeButton("취소", null)
dlg.show()
```

### 02 목록 대화상자

#### 기본 목록 (setItems)
- 항목 선택 시 대화상자가 자동으로 닫힘
```kotlin
var versionArray = arrayOf("오레오", "파이", "큐(10)")
dlg.setItems(versionArray) { dialog, which ->
    button1.text = versionArray[which]
}
```

#### 라디오버튼 목록 (setSingleChoiceItems)
- 항목 선택해도 대화상자가 닫히지 않음
- 두 번째 파라미터: 초기 선택 인덱스
```kotlin
dlg.setSingleChoiceItems(versionArray, 0) { dialog, which ->
    button1.text = versionArray[which]
}
```

#### 체크박스 목록 (setMultiChoiceItems)
- 여러 개를 동시에 선택 가능
```kotlin
var checkArray = booleanArrayOf(true, false, false)
dlg.setMultiChoiceItems(versionArray, checkArray) { dialog, which, isChecked ->
    button1.text = versionArray[which]
}
```

#### 실습 7-3: 사용자 정보 입력 앱 만들기

**기능 개요**
- 그림과 색상이 들어간 토스트 출력
- 사용자 정보(이름, 이메일)를 입력받는 커스텀 대화상자

**dialog1.xml (대화상자 레이아웃)**
```xml
<LinearLayout>
    <TextView android:text="사용자 이름" />
    <EditText android:id="@+id/dlgEdt1" />
    <TextView android:text="이메일" />
    <EditText android:id="@+id/dlgEdt2" />
</LinearLayout>
```

**toast1.xml (커스텀 토스트 레이아웃)**
```xml
<LinearLayout
    android:background="#ff0000"
    android:gravity="center">
    <ImageView android:src="@drawable/btn_star_big_on" />
    <TextView android:id="@+id/toastText1" android:textSize="20dp" />
    <ImageView android:src="@drawable/btn_star_big_on" />
</LinearLayout>
```

**<확인> 클릭 시 - 대화상자 에디트텍스트 내용을 메인 화면에 표시**
```kotlin
dlg.setPositiveButton("확인") { dialog, which ->
    dlgEdtName = dialogView.findViewById<EditText>(R.id.dlgEdt1)
    dlgEdtEmail = dialogView.findViewById<EditText>(R.id.dlgEdt2)

    tvName.text = dlgEdtName.text.toString()
    tvEmail.text = dlgEdtEmail.text.toString()
}
```

**<취소> 클릭 시 - 커스텀 토스트 출력**
```kotlin
dlg.setNegativeButton("취소") { dialog, which ->
    var toast = Toast(this@MainActivity)
    toastView = View.inflate(this@MainActivity, R.layout.toast1, null)
    toastText = toastView.findViewById<TextView>(R.id.toastText1)
    toastText.text = "취소했습니다"
    toast.view = toastView
    toast.show()
}
```

**커스텀 뷰가 있는 대화상자 생성**
```kotlin
button1.setOnClickListener {
    dialogView = View.inflate(this@MainActivity, R.layout.dialog1, null)
    var dlg = AlertDialog.Builder(this@MainActivity)
    dlg.setTitle("사용자 정보 입력")
    dlg.setIcon(R.drawable.ic_menu_allfriends)
    dlg.setView(dialogView)
    dlg.setPositiveButton("확인", null)
    dlg.setNegativeButton("취소", null)
    dlg.show()
}
```
